# 备份设置

## 备份内容

1. 配置文件 `config.json`
2. 安装的插件
3. 历史记录 `history.db`

## 本地备份

### 备份

1. 选择 `本地备份` 后点击备份按钮
2. 选择保存路径
3. pot会将所有备份内容打包为zip保存到选择的路径

### 恢复

1. 选择 `本地备份` 后点击恢复备份按钮
2. 选择备份文件
3. pot会将备份内容恢复到指定路径

## WebDav

### 备份

1. 选择 `WebDav`
2. 填写账号信息，所有支持WebDav的服务都是可以的
3. 填写完成后点击备份
4. pot会将所有备份内容打包后上传

### 恢复

1. 选择 `WebDav`
2. 填写账号信息，所有支持WebDav的服务都是可以的
3. 填写完成后点击恢复备份备份
4. 在备份列表中选择需要恢复的备份
5. pot会将备份文件下载后恢复到指定路径
